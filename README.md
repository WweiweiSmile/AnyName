# 一个自己想要的NAS

## 环境配置

go 版本 1.23.4
node 版本 v18.16.0
pnpm 版本 8.4.0

## 配置数据库

请在 Back/config.json 中配置数据库连接信息，config.json文件默认是不存在的，请自己创建

```json
{
  "name": "数据库名称",
  "username": "用户名",
  "password": "用户密码",
  "host": "数据库地址",
  "port": "端口"
}
```
